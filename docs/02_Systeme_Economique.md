# ⚙️ Système Économique Principal

Le cœur du jeu : une **bourse dynamique** basée sur l'offre et la demande.

## Instruments
- Achat/Vente Spot
- Contrats à terme (Futures)
- Vente à découvert (Short)
- Taxes et frais de marché

Les prix évoluent via les volumes échangés, les événements aléatoires et les lois économiques.
